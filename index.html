Programação

O que é o JavaScript?

Resposta

O JavaScript é uma linguagem de programação que permite criar conteúdo dinâmico em sites.

HTML

Para que serve a tag <main>?

Resposta

A tag <main> serve para indicar o conteúdo principal de uma página web.

CSS

O que faz a propriedade 'display: flex'?

Resposta

Ela ativa o Flexbox, facilitando o alinhamento e a distribuição de espaço entre os elementos.
